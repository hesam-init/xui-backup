# X-UI User Backup

## Requirements

- Node = 16+
- Git
- [Sqlite Studio](https://github.com/pawelsalawa/sqlitestudio/releases/tag/3.3.3)

## Setup

Clone the repo and install the dependencies.

```bash
npm install
```

```bash
yarn
```

## Config .env

Insert your x-ui panel info in .env file:

SERVER= ip or domain

PORT= panel port

COOKIE= cookie for auth
